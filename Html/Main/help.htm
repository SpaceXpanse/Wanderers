How to use Help

This help window can be opened from the SpaceXpanse Launchpad dialog by pressing the "Help" button, or during a simulation by pressing Alt F1, or selecting "Help" from the main menu (F4).

You can close the window by clicking the "X" button in the top right corner.

Double-clicking in the title bar will minimise the window. This feature keeps help at your fingertips without cluttering the 3-D simulation window. (The same trick also works for all other in-game dialog boxes.)

You can navigate the help system with the contents, index and search facilities on the left side of the help window (This area can be displayed or hidden by pressing the ("Show" or "Hide" button in the navigation bar at the top.)

Some scenarios come with context-sensitive help pages, which can be accessed by pressing the "Scenario" button in the navigation bar, if it is shown.

If the spacecraft you are currently controlling provides a help file, it can be accessed by pressing the "Vessel" button in the navigation bar.
