Orbit MFD controls

Keyboard Button Description
Shift A AR Auto-select reference object
Shift D DST Toggle display mode for distance parameters (radius vector, aphelion and perihelion distance: planetocentric radius (Rad, ApR, PeR) or altitude above planet mean radius (Alt, ApA, PeA).
Shift F FRM Toggle frame of reference (ecliptic, equator of reference object)
Shift M MOD Toggle display mode (list only, graphics only and both)
Shift N NT Unselect target
Shift P PRJ Toggle orbit projection mode (reference frame, ship's and target's orbital plane)
Shift R REF Select new reference object (planet or moon) for orbit calculation
Shift T TGT Open menu for target selection
