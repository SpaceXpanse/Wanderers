Transfer MFD controls

Keyboard Button Description
Shift R REF Open input box for selection of reference celestial body.
Shift S SRC Open a menu for source orbit selection.
Shift T TGT Open a menu for target selection.
Shift N NT Unselect target.
Shift X HTO Toggle HTO (hypothetical transfer orbit) display on/off.
Shift M NUM Toggle numerical multi-body trajectory calculation.
Shift U UPD Refresh numerical trajectory, if displayed.
Shift Z STP Open input box for time step definition.
Shift ,/. EJ-/EJ+ Rotate transfer orbit ejection longitude.
Shift -/= DV-/DV+ Decrease/increase ejection velocity difference.
