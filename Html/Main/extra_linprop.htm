Linear state propagators

This dialog provides advanced options for fine-tuning SpaceXpanse's dynamic state propagation method. It determines the algorithm SpaceXpanse uses to update vessel positions and velocities from one frame to the next. The options in this dialog are mainly for debugging and testing. Most users can keep the default settings.

It is possible to use up to 5 different propagation methods depending on the time interval per frame. For short intervals, simple (and fast) calculations such as RK-2 can be used, while for longer intervals more complex (and computationally expensive) methods such as RK-8 should be used.

For maximum fidelity (but potentially lower frame rates in the presence of a large number of spacecraft), high-order Runge-Kutta or symplectic integrators can be used also for shorter time intervals.

For details about the state integrators available in SpaceXpanse, see SpaceXpanse Technical Notes: Dynamic state vector propagation (in Doc\Technotes\dynamics.pdf).
