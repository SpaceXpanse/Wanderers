Flight recorder

The flight recorder allows to record simulation sessions for later replay. To open the flight recorder dialog, press Ctrl F5, or select Record from the main dialog (F4).

You can select a name under which the playback scenario will be saved. The default selection is the name of the scenario currently running. All playback scenarios are saved in the Playback folder. If you select a name that exists already, a confirmation message will pop up before overwriting the existing playback.

You can then press the REC button to start the recording. Recording will stop when you press the STOP button or end the simulation session.

To start the playback, simply launch the generated scenario under the Playback folder.

Keyboard shortcuts

You can start and stop recording without opening the recorder dialog by pressing Ctrl C.

Advanced recording options

Pressing the ↓ button on the recorder dialog opens additional recording options. These include:

Advanced recorder options

  • Record time acceleration: Record changes in time accelerations in the playback stream. This allows the playback to run at the same acceleration settings. If disabled, or if the user disables automatic time acceleration during playback, the acceleration setting can be changed manually during playback.
  • Record focus events: Record the change of input focus from one vessel to another in the playback stream. These switches will then also occur during playback. Otherwise, the input focus stays with the initial vessel throughout the playback, even if focus was switched during recording.
  • Sampling in system time steps: If selected, the system clock will be used to calculate the sampling intervals for saving position and attitude data. Otherwise, the simulation time will be used. If using system time, sampling intervals will be larger during time acceleration. This is usually preferred, because otherwise large amounts of redundant data may be recorded during coasting phases that are recorded at high time acceleration settings.

Attitude data

Allows to select the frame of reference to which the attitude data refer.

  • Ecliptic frame: Use the inertial ecliptic (J2000) frame of reference
  • Local horizon frame: Use the current horizon frame. In this case, the recorded Euler angles correspond directly to pitch, yaw and roll.

Position/velocity data

Allows to select the frame of reference for the linear state vectors (position and velocity).

  • Ecliptic fame: Use the ecliptic (J2000) frame of reference
  • Equatorial frame: Use the equatorial frame of the current orbit reference object.

For more details about playback file stream formats and data, see Doc\Technotes\RecorderRef.pdf.

Playback options

If the Recorder/playback dialog is opened during a replay, it allows to tune the replay options.

  • Show inflight notes: Show the screen annotations recorded with the playback stream.
  • Play at recording speed: Automatically switch time acceleration settings to the values recorded in the playback stream.
  • Use recorded camera settings: Automatically set the camera mode and position to the values recorded in the playback stream.
  • Use recorded focus events: Switch the input focus between vessels as recorded in the playback stream.

In addition, you can open the Playback editor by pressing the Editor button. The playback editor allows to enhance the playback stream by inserting annotations, camera positions, and time acceleration changes.
