Sync Orbit MFD controls

Keyboard Button Description
Shift T TGT Select target object. Only objects orbiting the same body as the ship will be accepted.
Shift M MOD Select reference axis mode. Intersection 1 and 2 are only available if the orbits intersect.
Shift N LEN Select number of lines in the orbit timing list.
Shift , R- Rotate reference axis (manual axis mode only).
Shift . R+ Rotate reference axis (manual axis mode only).
