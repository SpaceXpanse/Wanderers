SpaceXpanse > Main menu > Menu configuration

Menu and infobar configuration

The behaviour of the SpaceXpanse main menu and the info blocks can be configured with the menu configuration dialog. To open the dialog, right-click on the main menu with the mouse. (If the main menu is hidden, you can display it by pressing F4).

Menu bar

You can select to show, hide or auto-hide the menu bar. When auto-hidden, the menu will scroll into view when you move the mouse to the top edge of the simulation window. If the menu is hidden, it can be displayed again by pressing F4.

You can also adjust the opacity of the menu background, and whether to display the menu icons or labels only.

Info bars

The info bars are displays in the top left and right corners of the simulation window. The left info block shows the current camera target, camera mode, field of view and distance to target. The right info block shows date (ephemeris date and MJD day number), simulation run time and current time accelartion setting.

The configuration dialog offers some configuration options for the info blocks, including the show/hide/auto-hide setting.

Auxiliary info bars

Optional info blocks can be attached to the left and right info bars. These include a frame rate monitor, data on the current graphics load, and viewport parameters.

Action indicator

Select the way actions (pause, resume, record, playback) are indicated on screen. The indicator can be set to flash briefly on an action, to permanently display on pause/record/playback, or not to show at all.
