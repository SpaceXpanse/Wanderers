Sync Orbit MFD display

Move the mouse over the images to see a description of the different elements.

Target object.
Ref: Reference axis mode: A selectable axis for which timings are computed. Can be orbit intersection point 1 or 2 (if applicable), ship and target apoapsis and periapsis, and manual. The manual axis can be rotated.
RAnm: True anomaly of reference axis. The direction of the reference axis w.r.t. the ship's periapsis.
DLng: Longitude difference. The angle between ship and target as seen from the central body.
Dist: Distance between ship and target [m].
RVel: Relative velocity between ship and target [m/s].
DTmin: Time-of-arrival difference. The minimum time difference[s] between the ship's and target's arrival at the reference point for any of the listed orbits.
Time of arrival list: Orbit counter.
Time of arrival list: ship time to reference for each orbit. The closest match with the target timings is indicated in yellow.
Time of arrival list: target time to reference for each orbit. The closest match with the ship timings is indicated in yellow.
RInc: Relative inclination between current and target orbits.
Ship radius vector.
Target radius vector.
Reference axis.
Legend:
Green curve: Ship orbit trajectory
Green line: Ship radius vector
Yellow curve: Target orbit trajectory
Yellow line: Target radius vector
Bright green line: Reference axis.
