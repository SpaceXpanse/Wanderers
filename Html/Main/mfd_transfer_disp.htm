Orbit reference celestial body.
Src: Source object for which to calculate a transfer trajectory.

TrL: Current true longitude of target object [deg.] (ecliptic frame).

HTO: Hypothetical transfer orbit parameters.

TLi: True longitude of target intercept event [deg.] (ecliptic frame).
DTi: Time to target intercept event [s].
Num orbit: Numerical orbit calculation parameters.
Stp: Number of time steps calculated for numerical trajectory.
T: Time interval covered by numerical trajectory [s].
RInc: Relative inclination between source and target orbital planes.
Origin: Location of reference object.

Solid yellow line: Current target object radius vector.

Dashed yellow line: Target object radius vector at intercept event.
Solid grey line: Intercept position.

Solid green trajectory: Current orbit of source object.
Solid yellow trajectory: Orbit of target object.
