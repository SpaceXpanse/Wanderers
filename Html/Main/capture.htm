Capturing screenshots

You can copy SpaceXpanse screenshots to files or to the clipboard with the Capture frames dialog. The dialog can be opened with Ctrl PrtScn.

To save a screenshot, select the clipboard or a file name. File paths are relative to the SpaceXpanse root directory and should not contain the file extension. If the path does not exist, it is created automatically. For file output you can also select the image format and quality setting (if applicable).

Then click Take snapshot to save the SpaceXpanse snapshot. If the file name is a number, it will be incremented automatically with each screenshot taken.

You can also take sequences of screenshots by clicking Start recording. SpaceXpanse then writes out an image for each time step, or each n-th time step if a Skip frames setting > 0 is selected.

Warning: Writing out image sequences will severely degrade frame rates and is not suitable for interactive simulation sessions. It is best to create image sequences for simulation sessions that don't require user interaction or pre-recorded sessions.

Also be aware that recording an image sequence will create a large number of files and fill up large amounts of disk space.

Image sequences can be useful for creating high-quality movies. Various video editing programs can create video files from such image sequences. For video creation, it is also useful to set fixed time step intervals in the SpaceXpanse launchpad under Extra | Debugging options | Fixed time steps. For example, if you want to create a movie with 25 frames per second, use time steps of 0.04 seconds and set a frame skip of 0.
