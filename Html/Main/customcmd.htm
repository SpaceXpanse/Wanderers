Custom functions

SpaceXpanse supports plugin modules that allow to extend the core functionality of the simulator (see also Modules tab). Some of these may provide custom functions, such as dialog boxes, that can be accessed via the Custom functions dialog box. To open this dialog, press Ctrl F4 or select Function from the main menu (F4).

The dialog contains a list of all custom functions recognised by SpaceXpanse. If you can't find a particular function, make sure that the corresponding module has been activated.

Click on a function to see a short description below the list. To activate a function, double-click it or select it and press OK.

The standard SpaceXpanse distribution comes with a set of modules, such as a scenario editor, remote vessel control or MFD window. A lot more 3rd party plugin modules can be downloaded from the SpaceXpanse repositories on the web.
