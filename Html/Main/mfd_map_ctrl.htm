Map MFD controls

Keyboard Button Description
Shift R REF Open input box for reference planet/moon selection.
Shift T TGT Open a menu for target selection.
Shift K TRK Switch automatic vessel track mode on or off.
Shift Z ZM Toggle between global map view and 2x zoom.
Shift [ << Scroll map display left.
Shift ] >> Scroll map display right.
Shift - UP Scroll map display up (only in zoom mode).
Shift = DN Scroll map display down (only in zoom mode).
