Main menu

During the simulation, the main menu is displayed at the top edge of the simulation window. If it is not visible, it can be scrolled into view by pressing F4, or by moving the mouse towards the top edge of the simulation window.

The menu provides access to some of the most common dialogs and functions. Most of these are also available directly via keyboard shortcuts.

Menu Shortcut Description
Ship F3 Vessel selection. Take the controls of a different spacecraft.
Camera Ctrl F1 Camera control dialog. Select views, targets and camera modes.
Speed Ctrl F2 Simulation time acceleration control.
Pause Ctrl P Pause or resume the simulation.
Info Ctrl I Open the object info dialog. Contains data about vessels, celestial bodies and surface bases.
Function Ctrl F4 Open the Custom function dialog, to access extra functionality provided by addon modules.
Display Ctrl F9 Open the configuration dialog for Planetarium mode, force and object axis display.
Map Ctrl M Open a map diaplay showing current position, surface bases and nav radio transmitters.
Record Ctrl F5 Open the flight recorder/playback dialog for recording and replaying simulation sessions.
Help Alt F1 Open the Help window.
Save Ctrl S Save the current simulation state in the Quicksave scenario folder.
Capture frames CtrlPrtScn Save screen snapshots or image sequences to file or clipboard.
Exit Ctrl Q Quit the current simulation session and return to the Launchpad dialog.

The behaviour of the main menu and the info bar overlays can be configured with the Menu configuration dialog, which can be opened by right-clicking on the main menu.
