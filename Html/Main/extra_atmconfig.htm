Atmosphere configuration

This dialog box allows the selection of different atmosphere models for some celestial bodies.

Currently supported for Earth are the Jaccia71-Gill model and the NRLMSIS-00 model. For details see the SpaceXpanse Technical Notes: Earth Atmosphere Model (Doc\Technotes\earth_atm.pdf).

In addition, the legacy model from SpaceXpanse 2006 is supported for backward compatibility. However, this model is less accurate above altitudes of 100km, and doesn't support altitudes greater than 200km.

Atmospheric effects can be disabled by selecting "[None]".

Mars and Venus each only provide a single atmosphere model, which can be enabled or disabled with the dialog.

Other celestial bodies may define atmosphere models hard-coded in the driver module or configuration file. These do not appear in the Atmosphere configuration dialog.
