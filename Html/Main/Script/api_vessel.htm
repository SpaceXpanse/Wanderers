vessel: Vessel access

The vessel library contains functions for obtaining vessel handles. Vessel handles can be used to query and control vessels via vessel methods.

vessel.get_handle Obtain a vessel handle
vessel.get_focushandle Obtain a handle for the current focus vessel
vessel.get_interface Obtain a vessel class instance
vessel.get_focusinterface Obtain a class instance for the current focus vessel
vessel.get_count Obtain the number of vessels in the current simulation session

hvessel = vessel.get_handle(name)
hvessel = vessel.get_handle(idx)

Returns a handle for a vessel in the current simulation session.

Parameters:

name (string):vessel name
idx (int):vessel index (≥ 0)

Return values:

hvessel (handle):vessel handle, or nil if vessel was not found.

Notes:

This function can be invoked with a vessel name or a vessel index. Example:

-- list all vessels in the simulation session
n = vessel.get_count()
for idx = 0,n-1 do
  hvessel = vessel.get_handle(idx)
  term.out (oapi.get_objname(hvessel))
end

See also:

oapi.get_objhandle

hvessel = vessel.get_focushandle()

Returns a handle for the current focus vessel (the vessel receiving user input).

Return values:

hvessel (handle)focus vessel handle

v = vessel.get_interface(hvessel)
v = vessel.get_interface(name)
v = vessel.get_interface(idx)

Returns a class instance for a vessel.

Parameters:

hvessel (handle):vessel handle
name (string):vessel name
idx (int):vessel index (≥ 0)

Return values:

v:vessel object

Notes:

This function can be invoked with a vessel handle, a vessel name, or a vessel index. It returns a vessel instance that can be used to get or set vessel state parameters via vessel methods.

v = vessel.get_focusinterface()

Returns a class instance for the current focus vessel.

Return values:

v:focus vessel object

Notes:

This function returns a vessel instance that can be uses to get or set vessel state parameters via vessel methods.

n = vessel.get_count()

Returns the number of vessels in the current simulation session.

Return values:

n (int):vessel count

Notes:

This function can be used to set up a loop over all vessels in the simulation, for example in connection with vessel.get_handle or vessel.get_interface.
